I use this structure when I create a new python file. Helps me keep the parts separate and line-lenght down.

```py
#!/usr/bin/python3

'''
------------------------------------------------------------------------
    IMPORTS
------------------------------------------------------------------------
'''

# Standard library imports

# Third party imports

# Local application imports


'''
------------------------------------------------------------------------
    CLASSES
------------------------------------------------------------------------
'''


'''
------------------------------------------------------------------------
    FUNCTIONS
------------------------------------------------------------------------
'''


'''
------------------------------------------------------------------------
    SCRIPT
------------------------------------------------------------------------
'''


'''
------------------------------------------------------------------------
    __MAIN__
------------------------------------------------------------------------
'''
```py
